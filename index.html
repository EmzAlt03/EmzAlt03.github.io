Project Idea: Multi-Management Android Application

Project Idea: Overview

The utilization of mobile technology in professional and everyday applications is becoming increasingly apparent.
For my personal IT project, I wanted to explore the further implementation of smart devices such as phones and tablets in an everyday setting.
The project will involve the construction of a basic multi-management android application/tool  using MIT app inventor and/or various other means.
The application will allow the user to input personal, health related and situational information about themselves forming an individual profile.
This information can then be viewed, edited or analysed by the user.
The application is directed at allowing them to manage and express themselves through self-evaluation.

Project Idea: Motivations

My personal interest in constructing this particular android application originated from my personal struggles in life.
Throughout the past couple of years, social anxiety and a host of medical problems made it extremely difficult to manage my personal responsibilities and fulfill my aspirations in life.
An article published on Bizfluent by Wanda Thibodeaux More info, states that scheduling,
organising and simplifying tasks help in “contributing to the short-term and long-term goals of [an] organisation” as well as substantially “improve employee morale”.
This statement helps generate the idea that organizing, planning and reviewing your responsibilities can in fact, improve your ability to overcome difficulties,
satisfy goals while maintain good mental. The use of an application to keep record of personal information is cost efficient, easily accessible and confidential.

Project Idea: Details

In general, android applications contain a variety of features and basic structural elements which I hope to emulate for this project.
The application is directed toward individuals who find it difficult to manage everyday activities from work related assignments to their health and wellbeing.
Through the use of crucial software including Appy Pie and Figma,
I will be able to effectively design an elaborate and highly sophisticated application with the intent of successfully deploying it prior to given deadlines.

The structural components of the application will be constructed through basic programming techniques and templates.
Appy Pie is a software application which can be used to download pre-existing templates,
elements and components allowing for increased productivity and a reduction in completion time.
Through the use of this form of software, I will implement a minimalist design through suggested methods of styling,
categorization and compartmentalisation to ensure that all user types can successfully navigate and manipulate the applications array of features.

In itself, the application will contain three of the following sections: A home screen where the user can note anything that’s on their mind such including thoughts about world events,
current emotional state or their opinions about others.The home screen will also prompt the user on initial entry,
to input basic information about themselves which will be stored and employed in the profile building phase.
The secondary component of the application involves a settings menu in which the user can edit their profile information,
profile image, read the assistance and FAQ sections as well viewing future updates and changes.
The final component of the application involves a rudimentary timetabling system hosting upcoming events,
important dates including personalised recommendations based on your profile status with an ability to set your own goals and note down individual achievements.

Although still undecided, the application may utilize other features present within most modern-day mobile phones and software.
Some of these features may include SMS/messaging services to share personal goals and thoughts with others,
use of the internet services to import information or images to reduce need to fill out every component in the profile building section.
The application will employ a friend’s section where the user can import friends from their contacts list.
This feature will surely increase user engagement by providing a means of communication.
This feature may be implemented early on in the design process in order to build a basic link between the user and the application.

Overall, the application will include a variety of features, all of which will be carefully considered and directed towards the user’s experience.
Software and web-based services like Appy Pie and Figma will ensure that the application has a relatively refined structure,
well balanced design and layout along with significant functionality.
The applications relative complexity will require a significant level of understanding which will be obtained through basic tutorials,
videos and practical exercises. Nevertheless, the management application will condense many basic mobile applications into a single,
minimalistic and organised tool, allowing the user to manage all aspects of life in the palm of their hands.
